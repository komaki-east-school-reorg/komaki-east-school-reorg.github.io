【サイトファイル一覧】
小牧市東部（篠岡地区）学校再編計画 市民情報サイト
生成日：2026-05-24
======================================================

■ HTMLページ
index.html          サイトのトップページ。計画概要・現在の状況・今後のスケジュールをまとめたホーム画面。
about.html          学校再編計画の背景・目的・対象校・新校名などを詳しく解説する計画概要ページ。
schedule.html       フェーズ別タイムライン・月別カレンダー・主要イベント一覧を掲載するスケジュールページ。
council.html        市議会での審議内容（令和8年第1回定例会）を整理した議会審議ページ。
community.html      地域協議会・区長会への報告経緯と住民意見をまとめた地域協議会ページ。
voices.html         再編計画に対する賛成・反対・中立の住民意見をカード形式で紹介するページ。
faq.html            よくある質問と回答をアコーディオン形式で提供するFAQページ。
bus.html            スクールバスの運行予定・対象校・費用負担などを解説するスクールバス情報ページ。
map.html            再編対象の各学校の位置を地図で確認できる学校マップページ。

■ スタイル
css/style.css       サイト全体のデザイン・レイアウト・カラー・レスポンシブ対応を定義するスタイルシート。

■ JavaScript
js/i18n.js          言語JSONファイルを非同期取得してページ内テキストを多言語に切り替える国際化スクリプト。
js/main.js          ハンバーガーメニュー・カレンダー・日付自動ステータス更新・公式ニュース表示などのUIロジックを担うメインスクリプト。

■ データ
data/news.json              GitHub Actionsが定期取得した市公式サイトの最新お知らせを格納するJSONファイル。
data/official_pages/        市公式サイト各記事ページの本文テキストスナップショット（自動生成・手編集不可）。git diffで本文変更を検知するために使用。
data/events.json            カレンダーイベントデータ（日付キー＋8言語ラベル）。schedule.htmlのカレンダーが実行時にfetchして描画。イベントの追加・変更はこのファイルを編集する。
data/site-facts.json        公式ページのスラッグとサイト内反映箇所の対応表（手動管理）。更新検知Issueに「更新候補箇所」を載せるために使用。
data/i18n/ja.json           日本語（デフォルト言語）の翻訳テキストをすべて格納する言語ファイル。
data/i18n/en.json           英語の翻訳テキストを格納する言語ファイル。
data/i18n/pt.json           ポルトガル語（ブラジル）の翻訳テキストを格納する言語ファイル。
data/i18n/vi.json           ベトナム語の翻訳テキストを格納する言語ファイル。
data/i18n/tl.json           フィリピノ語（タガログ語）の翻訳テキストを格納する言語ファイル。
data/i18n/es.json           スペイン語の翻訳テキストを格納する言語ファイル。
data/i18n/zh.json           中国語（簡体字）の翻訳テキストを格納する言語ファイル。
data/i18n/id.json           インドネシア語の翻訳テキストを格納する言語ファイル。
data/i18n/ja-kids.json      「こどもむけ」モード用の上書き辞書。小学3年生レベルのやさしい日本語でja.jsonのキーを上書きする。

■ 画像・アイコン
favicon.svg                 ブラウザタブに表示されるサイトアイコン（SVG形式）。
img/og-image.png            SNSシェア時に表示されるOGP用サムネイル画像。

■ GitHub Actions
.github/workflows/fetch-news.yml    市公式サイトから最新お知らせを定期取得してnews.jsonと本文スナップショットを更新し、本文変更時はIssue起票→AIによる修正案の起案→機械検証→監査AI→自動マージまで行うワークフロー定義。
.github/scripts/fetch_news.py       市公式サイトをスクレイピングしてお知らせ情報の抽出・整形と各ページ本文のスナップショット保存を行うPythonスクリプト。
.github/scripts/map_targets.py      変更された公式ページのスラッグをsite-facts.jsonで引き、サイト内の更新候補箇所をMarkdown出力するスクリプト（Issue本文生成用）。
.github/scripts/auto_gates.py       自動更新のガードレール検証（編集範囲・スキーマ・外部リンク規則・出典実在チェック）。exit 0=合格/3=変更なし/1=不合格。
.github/scripts/auto_report.py      自動更新のPR本文・事後報告コメントをevidence.json/verdict.jsonから生成するスクリプト。

■ 設定・その他
.claude/settings.local.json    Claude Codeのプロジェクト固有の権限・設定を記述するローカル設定ファイル。
CONTRIBUTING.txt               コンテンツ管理チェックリスト。外部リンク制限・i18nルール・日付連動の自動表示・こどもむけ等の運用ルールと検証手順を記載。
README.md                      プロジェクト概要・ページ構成・対応言語・ローカル開発手順を説明する公開用READMEファイル。
CLAUDE.md                      Claude Code向けの開発ガイド（英語）。i18n仕様・検証コマンド・制約事項を記載。
CLAUDE_ja.md                   CLAUDE.mdの日本語説明ファイル。日本語話者向けにCLAUDE.mdの内容を解説。
files.txt                      本ファイル。サイト構成ファイルの一覧と説明を手動管理するインデックス。

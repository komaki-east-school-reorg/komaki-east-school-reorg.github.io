【サイトファイル一覧】
小牧市東部（篠岡地区）学校再編計画 市民情報サイト
生成日：2026-05-19
======================================================

■ HTMLページ
index.html          サイトのトップページ。計画概要・現在の状況・今後のスケジュールをまとめたホーム画面。
about.html          学校再編計画の背景・目的・対象校・新校名などを詳しく解説する計画概要ページ。
schedule.html       フェーズ別タイムライン・月別カレンダー・主要イベント一覧を掲載するスケジュールページ。
council.html        市議会での審議内容（令和8年第1回定例会）を整理した議会審議ページ。
community.html      地域協議会・区長会への報告経緯と住民意見をまとめた地域協議会ページ。
voices.html         再編計画に対する賛成・反対・中立の住民意見をカード形式で紹介するページ。
faq.html            よくある質問と回答をアコーディオン形式で提供するFAQページ。
bus.html            スクールバスの運行予定・対象校・費用負担などを解説するスクールバス情報ページ。
map.html            再編対象の各学校の位置を地図で確認できる学校マップページ。

■ スタイル
css/style.css       サイト全体のデザイン・レイアウト・カラー・レスポンシブ対応を定義するスタイルシート。

■ JavaScript
js/i18n.js          言語JSONファイルを非同期取得してページ内テキストを多言語に切り替える国際化スクリプト。
js/main.js          ハンバーガーメニュー・カレンダー・日付自動ステータス更新・公式ニュース表示などのUIロジックを担うメインスクリプト。

■ データ
data/news.json              GitHub Actionsが定期取得した市公式サイトの最新お知らせを格納するJSONファイル。
data/i18n/ja.json           日本語（デフォルト言語）の翻訳テキストをすべて格納する言語ファイル。
data/i18n/en.json           英語の翻訳テキストを格納する言語ファイル。
data/i18n/pt.json           ポルトガル語（ブラジル）の翻訳テキストを格納する言語ファイル。
data/i18n/vi.json           ベトナム語の翻訳テキストを格納する言語ファイル。
data/i18n/tl.json           フィリピノ語（タガログ語）の翻訳テキストを格納する言語ファイル。
data/i18n/es.json           スペイン語の翻訳テキストを格納する言語ファイル。
data/i18n/zh.json           中国語（簡体字）の翻訳テキストを格納する言語ファイル。
data/i18n/id.json           インドネシア語の翻訳テキストを格納する言語ファイル。

■ 画像・アイコン
favicon.svg                 ブラウザタブに表示されるサイトアイコン（SVG形式）。
img/og-image.png            SNSシェア時に表示されるOGP用サムネイル画像。

■ GitHub Actions
.github/workflows/fetch-news.yml    市公式サイトから最新お知らせを定期取得してnews.jsonを更新するワークフロー定義。
.github/scripts/fetch_news.py       市公式サイトをスクレイピングしてお知らせ情報を抽出・整形するPythonスクリプト。

■ 設定・その他
.claude/settings.local.json    Claude Codeのプロジェクト固有の権限・設定を記述するローカル設定ファイル。
CONTRIBUTING.txt               サイトへの情報提供・修正提案の方法を説明するコントリビューションガイド。
